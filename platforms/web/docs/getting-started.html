Getting Started

Build your first Zylix application in under 5 minutes. Learn the fundamentals of components, state, and rendering.

Quick Start

The fastest way to get started is with a simple HTML file. No build tools required!

1
Create an HTML file

Create a new file called index.html:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Zylix App</title>
</head>
<body>
  <div id="app"></div>
  <script type="module" src="app.js"></script>
</body>
</html>
2
Create your app

Create app.js with your first component:

import { h, render, useState } from './zylix.js';

// A simple counter component
function Counter() {
  const [count, setCount] = useState(0);

  return h('div', { style: 'text-align: center; padding: 40px;' },
    h('h1', null, 'Counter'),
    h('p', { style: 'font-size: 3rem; margin: 20px 0;' }, count),
    h('button', {
      onClick: () => setCount(c => c + 1),
      style: 'padding: 10px 20px; font-size: 1rem; margin: 5px;'
    }, 'Increment'),
    h('button', {
      onClick: () => setCount(c => c - 1),
      style: 'padding: 10px 20px; font-size: 1rem; margin: 5px;'
    }, 'Decrement')
  );
}

// Render the app
render(h(Counter), document.getElementById('app'));
3
Run your app

Serve your files with any static server:

# Using Python
python -m http.server 3000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:3000

Open http://localhost:3000 in your browser!

Tip

You can also try our Hello World sample to see Zylix in action without any setup.

Understanding Components

Components are the building blocks of Zylix applications. They are functions that return virtual DOM nodes using the h() function.

The h() Function

The h() function creates virtual DOM elements. It takes three arguments: 

h(type, props, ...children)

// Examples:
h('div', null)                        // <div></div>
h('p', null, 'Hello')                 // <p>Hello</p>
h('button', { onClick: fn }, 'Click') // <button onclick="fn">Click</button>
h('ul', null,
  h('li', null, 'Item 1'),
  h('li', null, 'Item 2')
)                                     // Nested elements

Functional Components

Components are just functions that return h() calls: 

// Simple component
function Greeting({ name }) {
  return h('h1', null, `Hello, ${name}!`);
}

// Use it
h(Greeting, { name: 'World' })  // <h1>Hello, World!</h1>

Component Composition

Build complex UIs by composing smaller components: 

function Button({ onClick, children }) {
  return h('button', {
    onClick,
    style: 'padding: 10px 20px; border-radius: 6px;'
  }, children);
}

function Card({ title, children }) {
  return h('div', { className: 'card' },
    h('h2', null, title),
    h('div', { className: 'card-body' }, ...children)
  );
}

function App() {
  return h(Card, { title: 'Welcome' },
    h('p', null, 'This is a card component.'),
    h(Button, { onClick: () => alert('Clicked!') }, 'Click Me')
  );
}

State Management

Use the useState hook to add state to your components.

Basic State

function Counter() {
  const [count, setCount] = useState(0);

  return h('div', null,
    h('p', null, `Count: ${count}`),
    h('button', { onClick: () => setCount(count + 1) }, '+'),
    h('button', { onClick: () => setCount(count - 1) }, '-')
  );
}

Functional Updates

When the new state depends on the previous state, use a function: 

// Safer for async updates
setCount(prevCount => prevCount + 1);

Object State

function Form() {
  const [form, setForm] = useState({ name: '', email: '' });

  const updateField = (field, value) => {
    setForm(prev => ({ ...prev, [field]: value }));
  };

  return h('form', null,
    h('input', {
      value: form.name,
      onInput: (e) => updateField('name', e.target.value),
      placeholder: 'Name'
    }),
    h('input', {
      value: form.email,
      onInput: (e) => updateField('email', e.target.value),
      placeholder: 'Email'
    })
  );
}

Effects and Side Effects

Use useEffect for side effects like data fetching, subscriptions, or DOM manipulation. 

function UserProfile({ userId }) {
  const [user, setUser] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    setLoading(true);

    fetch(`/api/users/${userId}`)
      .then(res => res.json())
      .then(data => {
        setUser(data);
        setLoading(false);
      });
  }, [userId]); // Re-run when userId changes

  if (loading) {
    return h('p', null, 'Loading...');
  }

  return h('div', null,
    h('h1', null, user.name),
    h('p', null, user.email)
  );
}

Cleanup Effects

Return a function from useEffect to clean up:

useEffect(() => {
  const timer = setInterval(() => {
    console.log('Tick');
  }, 1000);

  // Cleanup function
  return () => clearInterval(timer);
}, []);
