Introduction

Zylix is a modern UI framework designed for building high-performance, cross-platform applications. It combines the developer experience of React with the performance characteristics of native applications.

Why Zylix?

  • Performance First: Built from the ground up for speed with a Zig-powered core
  • Familiar API: React-like hooks and component model reduces learning curve
  • Cross-Platform: Single codebase targets Web, iOS, Android, and Desktop
  • Developer Experience: DevTools, HMR, and comprehensive testing utilities
  • Production Ready: Battle-tested with real-world applications

Quick Start

Create your first Zylix component in seconds:

import { h, render, useState } from 'zylix';

function Counter() {
  const [count, setCount] = useState(0);

  return h('div', {},
    h('p', {}, `Count: ${count}`),
    h('button', { onClick: () => setCount(c => c + 1) }, 'Increment')
  );
}

render(h(Counter), document.getElementById('app'));

Core Features

Reactive Components

Build interactive UIs with reactive state management:

function TodoList() {
  const [todos, setTodos] = useState([]);
  const [input, setInput] = useState('');

  const addTodo = () => {
    if (input.trim()) {
      setTodos([...todos, { id: Date.now(), text: input }]);
      setInput('');
    }
  };

  return h('div', {},
    h('input', {
      value: input,
      onInput: e => setInput(e.target.value)
    }),
    h('button', { onClick: addTodo }, 'Add'),
    h('ul', {},
      ...todos.map(todo => h('li', { key: todo.id }, todo.text))
    )
  );
}

Global State with Store

Manage complex application state with the built-in store:

const store = createStore({
  state: { count: 0, user: null },

  actions: {
    increment: (state) => ({ ...state, count: state.count + 1 }),
    setUser: (state, user) => ({ ...state, user })
  },

  selectors: {
    isLoggedIn: (state) => state.user !== null,
    doubleCount: (state) => state.count * 2
  }
});

// Use in components
function Counter() {
  const count = useStore(store, s => s.count);
  return h('button', { onClick: store.actions.increment }, count);
}

HTTP Client

Fetch data with the built-in HTTP client:

const client = createHttpClient({
  baseURL: 'https://api.example.com',
  headers: { 'Authorization': 'Bearer token' }
});

// GET request
const users = await client.get('/users');

// POST with automatic JSON handling
const newUser = await client.post('/users', {
  name: 'John',
  email: 'john@example.com'
});

Testing

Zylix includes a comprehensive testing library inspired by @testing-library/react: 

import { render, screen, fireEvent, waitFor } from 'zylix/testing';

test('counter increments', async () => {
  render(h(Counter));

  expect(screen.getByText('Count: 0')).toBeInTheDocument();

  fireEvent.click(screen.getByText('Increment'));

  expect(screen.getByText('Count: 1')).toBeInTheDocument();
});

test('async data loading', async () => {
  render(h(UserProfile));

  // Wait for loading to complete
  await waitFor(() => {
    expect(screen.getByText('John Doe')).toBeInTheDocument();
  });
});
