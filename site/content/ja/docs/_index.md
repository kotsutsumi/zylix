Zylix ドキュメントへようこそ！Zig でクロスプラットフォームアプリケーションを構築する方法を学びましょう。

## Zylix とは？

Zylix は Zig で構築された高性能クロスプラットフォーム UI フレームワークです：

- **Virtual DOM**: 効率的なレンダリングと最小限の更新
- **6 プラットフォーム対応**: Web、iOS、Android、macOS、Linux、Windows
- **C ABI**: ネイティブプラットフォームフレームワークとの簡単な統合
- **型安全**: コンパイル時保証による信頼性
