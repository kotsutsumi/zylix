Welcome to the Zylix documentation! Learn how to build cross-platform applications with Zig.

## What is Zylix?

Zylix is a high-performance, cross-platform UI framework powered by Zig. It provides:

- **Virtual DOM**: Efficient rendering with minimal updates
- **6 Platform Support**: Web, iOS, Android, macOS, Linux, Windows
- **C ABI**: Easy integration with native platform frameworks
- **Type Safety**: Compile-time guarantees for reliability
